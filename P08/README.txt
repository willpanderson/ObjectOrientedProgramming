Directions to run the Mav's Animal Shelter Software for those whomever it may concern . 


1. When the program opens, the sensitivity is set to the client mode. 
     
    To access the diffrent levels, select Role>(Client,Staff,Manager,Director) and enter in the respective password.

    Client: No password here, automatically set when button is clicked
    Manager: "manager"
    Staff: "staff"
    Director: "director"

4. There are four types of files that can be used as test cases for file handling included. 
    
   -editere.mass (file with a name that has no adoptions)
   -untitled.mass (for simply saving with no name and listing clients with adoptions)
   -namey.muss (invalid file type)
   -nameyx.mass (missing version and cookie in order to properly load)

5. When a new file is loaded into the shelter or when a new shelter is made, the sensitivity defaults to client mode.
